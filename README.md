<h1>Products Aggregator</h1>
<p>Products Aggregator is a web application that aggregates product information from multiple online stores, such as Amazon, eBay, AliExpress, Jumia, and others. The application allows users to easily compare products across different stores in one place and save them for later.</p>
<h2>Features</h2>
<ul>
  <li>Scrapes product information from multiple online stores.</li>
  <li>Returns detailed reports about each product, including prices, availability, reviews, and more.</li>
  <li>Allows users to easily compare products across multiple stores.</li>
  <li>Saves products for later.</li>
  <li>User-friendly interface with advanced search and filtering options.</li>
</ul>
<h2>Technologies Used</h2>
<ul>
  <li>Python</li>
  <li>BeautifulSoup (Python library for web scraping)</li>
  <li>Django (Python web framework)</li>
  <li>PostgreSQL (Relational database)</li>
  <li>HTML/CSS/JavaScript (Front-end technologies)</li>
</ul>
<h2>How to Use</h2>
<ol>
  <li>Clone the repository to your local machine.</li>
  <li>Install the required dependencies listed in the <code>requirements.txt</code> file.</li>
  <li>Set up your PostgreSQL database and run the migrations.</li>
  <li>Run the server.</li>
  <li>Browse the available endpoints and interact with the API using your preferred client (e.g., Postman).</li>
</ol>
